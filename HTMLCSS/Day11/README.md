2. Responsive

- Media Queries: A CSS tool that helps define "breakpoints" to change formatting (e.g., hiding content, changing font size) when the screen is smaller
- Flexible Grid Layout: A column-based system that allows web elements to rearrange themselves (for example, from 3 columns on a computer to 1 column on a phone).
- Flexible images: Photos and videos will automatically resize to avoid overflowing the screen or appearing distorted.(kh bị tràn và vỡ)
