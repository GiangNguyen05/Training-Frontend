### TRAIN FRONTEND

Responsive:

- Create a demo on a mobile screen.
  - Screen: single column layout
    - 320px – 480px: Standard smartphone screen (vertical display).
    - 480px – 640px: Larger smartphone screen or horizontal display.
    - 640px – 768px: Smaller lines of a computer table.
  - Cores:
    - tag meta viewport: This lets the browser know how to adjust the display scale for mobile devices.(điều chỉnh)
    - Media Queries: CSS tools for different formatting on specific screens.(định dạng)
    - Use relative units instead of px, such as: %, vw, rem, em,...
