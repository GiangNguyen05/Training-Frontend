   Flex: display: flex;

- One way layout (row or col): Create a simple interface structure.(giao diện đơn giản)
- This can be used when the child elements have variable sizes(không cố định)
- It provides a container that allows you to change the elements: width, height, and display order to fill the available space
- Useful for elements such as navigation bars, menus, etc., suitable(thích hợp) for responsive design.

  Grid: display:Grid;

- It can be arranged in two directions (row and cols):creating complex structures(phức tạp)
- We often need to determine the precise and tight positions of the elements beforehand(xác định ele cố định)
