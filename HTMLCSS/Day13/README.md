# TRAIN FRONTEND

## Responsive

- Create a demo on a tablet screen.
  - Screen: 2-3 column layout
    - 640px - 768px: tablet mini(7-8inch).
    - 768px – 1024px: Larger tablet screen.
    - Above 1024px: Large-screen tablets or tablets in landscape mode – the interface is usually similar to a laptop.

  - Cores:
    - Layout conversion: Use a 2-3 column layout (avoid a sparse layout)(thưa thớt).
    - Navigation menu: The tablet is wide enough to display a tab bar or sidebar menu.
    - Images size: High quality -> avoid blurring and revealing flaws(mờ, lộ khuyết điểm)
