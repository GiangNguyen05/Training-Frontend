The <video> element: To play and control the video, you can use the <video> tag and the Web Video API (controls the player such as play, pause, and skip to the next video).
The <video> attribute:

- src: source of the video.
- width and height: size of the video
- controls: If this boolean attribute is present, the browser displays its own controls for video playback and volume.
- poster: This attribute allows you to specify an image that the browser will use while video is being downloaded, or until the user starts playing the video.
- autoplay: This attribute asks the browser to start playing the video automatically as soon as the page is ready.
- preload: The preload attribute is used when autoplay is not used.
- loop: Another boolean attribute that indicates to play the video in loop mode(lặp lại hay kh)

The <audio> element: To handle audio, you can use the audio tag (in HTML5) and the WebAudio API (not yet fully understood)

The <audio> attribute:

- src: The path to the audio file (mp3, wav, ogg...).
- controls: Displays the default browser controls (Play, Pause, volume, progress bar). If this is missing, the sound detector will be hidden.
- autoplay: Automatically plays music as soon as the webpage finishes loading (Note: Most browsers nowadays will block autoplay if the user hasn't interacted with the page).
- loop: Automatically restarts playback from the beginning.
- mute: By default mutes playback.
