- Using semantic tags, ID attributes, and classes. Apply pseudo-classes.
  -->section tag:
  - Semantic meaning helps search engines understand the page structure and break down the page content.
  - Always include headings (h1-h6)
  - Do not use as a replacement(thay thế) (do not use solely for CSS formatting without conveying a distinct thematic meaning)
    -->article tag:
  - Used to enclose independent(bao hàm nội dung độc lập), self-contained content that has its own distinct meaning(ý nghĩa riêng biệt)
  - It can be reused or shared independently(kh phụ thuộc) of the rest of the page
