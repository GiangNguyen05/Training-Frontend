Layout

- how to lay out a webpage
- use position properties
  Relative: Precisely position the components without affecting their original positions or other components
  Absolute: Identify components by parent tag(Relative nếu có),Without a parent tag, it will follow the body of the page.
  Fixed: Used to position an element relative to the display window of browsers.(không thay đổi)
- Use grids and floats to align elements for page layout.
